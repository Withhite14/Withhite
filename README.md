### üèÜ Featured Projects

#### ü§ñ **EIS Assistant (Enterprise GenAI Agent)**
*A smart AI agent designed to democratize data access for business users.*
* **Problem:** Business teams relied on manual ad-hoc SQL requests to Data Engineers for every report, causing bottlenecks.
* **Solution:** Built an **Agentic RAG** system using **LangGraph** and **OpenAI API**.
    * **Router Architecture:** Dynamically routes queries between a **Vector Store** (for document retrieval) and a **Text-to-SQL** engine (for database queries).
    * **Tech Stack:** Python, LangChain, Oracle DB, Streamlit.
* **Impact:** Reduced ad-hoc data requests by **~30%**, enabling self-service analytics.

#### ‚ö° **Project SilverStream (Data Pipeline Modernization)**
*A high-performance data extraction prototype migrating from legacy ETL to Spark.*
* **Context:** Legacy Informatica workflows were struggling with scalability and licensing costs.
* **Implementation:** Developed a POC using **Apache Spark (PySpark)** to handle large-scale data extraction from Oracle.
* **Outcome:** Demonstrated potential for significantly faster data processing and laid the groundwork for the organization's shift to a Python-based Modern Data Stack.

#### üß† **Master's Thesis: Human Pose Estimation & Early Action Prediction**
*Novel research in Computer Vision accepted for publication at **ITC-CSCC 2024**.*
* **Objective:** Predict human actions *before* they are completed to enable proactive robotic responses.
* **Method:** Proposed a Deep Learning architecture utilizing **Progressive Skeleton Data**.
* **Result:** Achieved superior accuracy compared to baseline models in early-stage detection.
